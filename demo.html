📚 Граждански процесуален кодекс

Интерактивна уеб апликация за визуализация на ГПК като mind map с препратки между членовете. Проектът включва автоматично откриване на преки и семантични връзки между членовете на закона.

🗺️ Интерактивна карта

Визуализация на всички членове като mind map с възможност за приближаване, отдалечаване и навигация.

🔗 Преки препратки

Автоматично откриване и показване на директните препратки между членовете (напр. "чл. 5", "ал. 2").

🧠 Семантични връзки

Интелигентно откриване на семантично свързани членове базирано на ключови думи и теми.

🔍 Търсене

Пълнотекстово търсене по номер на член, заглавие и съдържание с маркиране на резултатите.

📱 Responsive дизайн

Адаптивен интерфейс, който работи отлично на всички устройства - десктоп, таблет и мобилен.

⚡ Бърза навигация

Кликване на препратки в текста за моментален преход между свързани членове.

🚀 Как да стартирам приложението?

1. Инсталирайте Node.js

curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash - sudo apt-get install -y nodejs

2. Инсталирайте зависимостите

cd gpk-mindmap npm install

3. Стартирайте приложението

npm run dev

Приложението ще се отвори автоматично на http://localhost:3000
